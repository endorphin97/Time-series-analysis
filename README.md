## 数据集来源
Kaggle 数据集 “Jena Climate Dataset” 

## 目的
针对德国耶拿地区 2009 年至 2016 年的气象时间序列数据，采用多种时间序列模型对摄氏温度（T_degC）进行建模与预测分析，旨在为区域气候变化研究及气象应用提供科学依据。

## 方法
预处理，筛选出 8 个关键特征（如大气压力、相对湿度、风速等），并按时间划分训练集（2009-2013 年）、测试集（2014-2015 年）和验证集（2016 年）。通过探索性数据分析（EDA）和 ADF 检验证实数据具有平稳性，Ljung-Box 检验拒绝纯随机性假设，表明数据适合建模。
在模型构建阶段，研究对比了多种模型的性能：ARIMA 模型：自动拟合结果显示 AIC/BIC 较低，但预测均方误差（MSE）为 178.46，表明单独使用 ARIMA 模型预测效果有限；SARIMA 与 Holt-Winters 模型：通过时间序列分解识别季节性和趋势特征，Holt-Winters 模型的 MSE（30.76）略优于 SARIMA（31.71），二者均能有效捕捉温度的周期性变化；ARIMAX 与 SARIMAX 模型：引入外生变量（如大气压力、湿度等）后，模型性能显著提升。ARIMAX 模型 MSE 降至 0.021，而 SARIMAX 通过构造正余弦季节性外生变量进一步优化，MSE 达到 0.0197，成为预测效果最佳的模型。

## 结果
结合外生变量和季节性特征的 SARIMAX 模型能更精准地捕捉耶拿地区温度变化规律，为农业、能源规划等领域应对极端天气提供了可靠的预测工具。未来可进一步拓展模型以纳入更多环境变量，提升复杂气候场景下的预测能力。
